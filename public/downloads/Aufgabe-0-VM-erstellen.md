# Aufgabe 0 – Virtuelle Maschine (VM) erstellen

Bevor du den Webserver (Aufgabe 1) aufsetzt, brauchst du eine VM.

> Platzhalter-Anleitung. Der Berufsbildner ersetzt diese Datei durch die
> offizielle PDF «Schnuppertage Aufgabe 0 VM Erstellen.pdf».

## Schritte (Hyper-V)

1. **Hyper-V Manager** öffnen.
2. **Neu → Virtueller Computer** wählen.
3. **Generation 2** auswählen.
4. **Arbeitsspeicher** zuweisen (z. B. 4096 MB).
5. **Netzwerk** verbinden (virtueller Switch).
6. **Virtuelle Festplatte** erstellen (z. B. 40 GB).
7. **ISO einbinden** (Installationsabbild).
8. Bei Bedarf **Secure Boot deaktivieren** und **Bootreihenfolge** anpassen.
9. VM **starten** und der Installation folgen.

## Checkliste
- [ ] VM startet
- [ ] Netzwerk funktioniert
- [ ] Betriebssystem installiert
